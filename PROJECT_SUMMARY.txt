HOW TO USE
================================================================================

1. Open index.html in a browser to see the demo
2. Copy simple-loader.js and simple-loader.css to your project
3. Include in HTML:
   <link rel="stylesheet" href="simple-loader.css">
   <script src="simple-loader.js"></script>
4. Create and use:
   const loader = new SimpleLoader({ type: 'spinner' });
   loader.show();

See QUICK_START.md for detailed instructions.
